# TP3_Partie1-Page-de-recherche-de-google
 🌙 Google Clone - HTML & CSS

 📌 Description

Ce projet est une reproduction de la page d'accueil de Google  .

L'objectif de ce projet est de pratiquer :
- La structure HTML
- Le positionnement avec Flexbox
- Le design responsive
- Le stylage moderne avec CSS
- La reproduction d'une interface réelle

---

🛠️ Technologies utilisées

- HTML5
- CSS3
- Flexbox

---

 📂 Structure du projet


google-clone/
│
├── index.html
└── style.css


---

🚀 Fonctionnalités

✔ Interface   
✔ Barre de recherche arrondie  
✔ Bouton Mode IA  
✔ Section raccourcis (YouTube, Ajouter un raccourci)  
✔ Header avec Gmail et Images  
✔ Design centré avec Flexbox  

---

 🎯 Objectif pédagogique

Ce projet permet de :

- Comprendre la structure d'une page web
- Utiliser Flexbox pour centrer des éléments
- Travailler les bordures arrondies
- Manipuler les couleurs et les effets hover
- Reproduire une interface existante
